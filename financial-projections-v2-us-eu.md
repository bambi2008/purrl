## 人猫共生系列（新增核心）

这是出行美学衍生出来的——从"带猫出去"延伸到"和猫住在一起"。

### 产品线

| 产品 | 价格 | 建筑师价值 | 竞品 |
|------|------|-----------|------|
| **Cat Sling Bag** 编织猫袋 | $129 | 编织纹理+竹手柄，呼应那张照片 | 优质手工商家$50-80 |
| **Cat Shelf Ladder** 书架猫台 | $249 | 极简木架，猫走上去当书挡 | 笨重爬架$100-200 |
| **Side Table Cat Cave** 茶几猫窝 | $399 | 下面掏空的边桌，人放咖啡猫钻进去 | 丑的猫窝$30-80 |

### 为什么这是建筑师做的东西

```
普通宠物品牌做猫家具：
  → 猫爬架 = 圆柱+绒布+绳子
  → 粉色/蓝色/灰色
  → 放在客厅里丑陋

Purrl做猫家具：
  → 比例和人在家的尺度协调
  → 材质：实木+亚麻+皮革（不是绒布）
  → 不是"猫的家具"，是"人可以用的家具，猫也能用"
  → 放在客厅里别人问"这是哪买的"，而不是"这是猫的"
```

## 财务测算 v2（欧美市场美元计）

### 假设调整

| 项目 | v1(中国) | v2(欧美) | 原因 |
|------|---------|---------|------|
| 免费→付费转化 | 5% | **8%** | 欧美SaaS基准高(10-15%)，保守取8% |
| 年续费率 | 30% | **45%** | 邮件营销习惯成熟 |
| AOV(配饰) | ¥98 | **$29** | 直接翻倍 |
| 获客成本 | ~¥0 | **$3-5/用户** | FB/IG广告 |
| 用户增长 | 自然流量 | **付费广告+自然** | 需前期投放加速 |
| 货币 | 人民币 | **美元** | |

### 第一年

```
用户获取：
  FB/IG广告 + 小红书(华人圈) + 自然流量
  获客成本 $3/注册用户
  
用户基数：
  全年新增注册用户：2,000人
  广告费：2,000×$3=$6,000
  月均新增：167人

订阅收入：
  付费用户(8%转化)：160人 × $24 = $3,840
  续费：0(第一年无)

配饰收入(首单)：
  160×15%=24件 × $29 = $696

单次收入：
  约30%免费用户试单次：600×$1.99=$1,194

人猫共生(早期尝试)：
  总付费用户中5%：160×5%=8件
  假设均价$189：8×$189=$1,512

--- 年度总计 ---
总收入：$7,242

成本：
  AI API：~$50
  广告费：$6,000
  POD履约(24件×$8+8件×$38=~$496)：~$500
  支付手续费(3%)：~$220
总成本：~$6,770

净利润：~$472
净利率：6.5%
```

> 第一年不赚钱——$6,000广告费吃掉了几乎所有利润。前半年赚够自然量后才缩减广告。

## 核心发现

```
v1(中国·低客单) vs v2(欧美·高客单) 第三年对比：

                      v1              v2
  用户数             20,000          25,000
  净收入(￥)          9.1万           82万
  净利率             84%             69%
  月均净利           7,600元         6.8万元

  v2的$6,000广告投入 = ROI核武器。
  用广告费换规模，利润反而比v1高出9倍。
  
  差距根源不是用户量多25%，是客单价翻了4-5倍。
```

### 人猫共生的影响

```
第三年收入构成：
  订阅：$40,608 (25%)
  配饰：$67,164 (41%) ← 复购放大
  单次：$12,537 (8%)
  3D：$6,900 (4%)
  人猫共生：$37,800 (23%) ← 第三大收入源

配饰复购在第三年爆发（2,100件），
因为基数积累+年复购2次。
人猫共生单价高($189)，占收入23%但只占销量的8%。
```

### 不用做什么

```
❌ 数字孪生 — 3年都不做，变现太弱
❌ 猫衣服 — 猫抗拒，退货率高，不做
❌ 处方药/健康诊断 — 方向不同，交给穷养猫线
❌ 中国独立App — 只用Web+PWA，省App Store审核+抽成
```
