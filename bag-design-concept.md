# Purrl 猫袋版型概念

> 参考照片：银虎斑猫坐在白色针织袋中，竹提手，敞口软结构

---

## 核心版型

```
         ┌─────────────┐
         │  收口（非抽绳）│     ← 比参考照片更收，但不封死
         │  猫头+前爪伸出 │
         │               │
    ─────┤   猫身体      ├─────    ← 竹/木提手环（对称两侧）
         │               │
         │   底部加宽    │
         └───────────────┘
         上窄下宽的梯形
```

## 参考照片的关键结构

| 特征 | 参考照片 | Purrl 要做的变化 |
|------|---------|----------------|
| 开口 | 完全敞开 | **有一定收口**（不是抽绳，而是结构收窄）|
| 提手 | 竹制圆环 | 同样的**天然材质提手**（竹/木/麻绳编）|
| 面料 | 白色针织（弹性） | **亚麻**（无弹性，靠剪裁实现收口）|
| 底部 | 软底，猫坐进去自然下坠 | 同样**软底**，加厚内衬加强承重 |
| 猫的姿势 | 头+前爪伸出，身体在里面 | 同样 |
| 整体结构 | 上窄下宽 | 同样梯形剪裁 |

## "有收口但不收死"的实现

亚麻没有弹性，所以不能靠面料自然收拢。两种方案：

### 方案 A：剪裁收口

```
   ┌─── 窄口（约20cm）───┐
   │                      │      ← 猫头+前爪伸出
   ╲                    ╱
    ╲                  ╱       ← 两侧斜裁，自然收窄
     ╲                ╱
      ╲              ╱
       ┌────────────┐        ← 底部加宽（约30cm）
```

- 袋口宽度约 20-22cm（猫头+前爪通过，身体不会滑出）
- 底部宽度约 30-32cm（猫坐得舒服）
- 深度约 35-40cm（猫的体长）
- 两侧斜裁形成梯形

### 方案 B：自然垂坠收口（参考照片方式）

- 袋口较宽（约28cm），但亚麻材质较软，猫坐进去后袋口自然垂坠变小
- 内层可加一条**可调节的暗扣带**——扣上时袋口收窄，解开时敞开

## 提手设计

参考照片的竹制圆环提手是点睛之笔。Purrl 可以用：

| 材质 | 效果 |
|------|------|
| **竹制圆环**（参照照片） | 自然色，跟亚麻搭配和谐，环保 |
| **哑光木制圆环** | 更精致，深色木跟浅色亚麻对比 |
| **麻绳编织提手** | 跟亚麻同源，更柔软但耐用性稍弱 |

**推荐：竹/木制圆环提手**，跟参考照片同风格但哑光处理。
