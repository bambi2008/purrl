## 一、供应链拆解

一个 Purrl Tote 需要来自 5 类不同供应商的部件：

```
亚麻面料 → 箱包缝制厂 → Purrl Tote
竹制提手 ↗
黄铜五金 ↗
有机棉织带 ↗

三角巾 = 亚麻面料 → 服装配件厂 → Bandana
项圈 = 棉织带 + 黄铜扣 → 宠物配件组装厂 → Collar
斜挎肩带 = 有机棉宽织带 → 织带厂 → Shoulder Strap
```

## 二、1688 搜索关键词

打开 https://www.1688.com，搜以下关键词。每个品类找 **2-3 家**有样品/小批量选项的厂，联系询价。

### 1. 亚麻袋缝制厂（最核心的厂）

| 关键词 | 说明 |
|--------|------|
| `亚麻托特包 定制` | 通搜 |
| `亚麻帆布袋 定做` | 可能叫帆布袋的厂也做亚麻 |
| `黄麻袋 定制` | 黄麻跟亚麻工艺接近，厂一般两种都做 |
| `亚麻布艺包 小批量` | 加"小批量"过滤只做大批发的 |

**重点看**：绍兴、嘉兴、南通、苏州的厂商。这些地区是麻纺产业带。

**询价时要说的事**：
- 数量：先打样 2-3 只，后续 50-100 只起订
- 面料：欧洲亚麻 220-260 gsm，厂家通常有现成面料可提供色卡
- 版型：梯形软袋，需提供尺寸图
- 工艺：法式缝（French seam），内侧无毛边
- 附加：袋内有小口袋、可拆卸五金套扣

### 2. 竹制提手

| 关键词 | 说明 |
|--------|------|
| `竹制手提环 包配件` | 通搜 |
| `竹圆环 箱包提手` | 精搜 |
| `竹木工艺 提手配件` | 换个角度 |
| `天然竹 圆形提手 定制` | 定制规格 |

**规格**：外径约 14-16cm，截面直径 18-22mm，哑光原色。需要开孔或预留孔用于连接袋身。

**重点地区**：安吉（浙江）——全国竹制品中心。

### 3. 黄铜五金

| 关键词 | 说明 |
|--------|------|
| `黄铜铆钉 箱包配件` | 固定提手用 |
| `黄铜扣环 箱包` | D 环/锚点环 |
| `黄铜套扣 可拆卸` | 提手可拆卸用 |
| `黄铜日字扣 调节扣` | 肩带调节 |

**重点地区**：温州、汕头、广州花都。

### 4. 有机棉织带 + 项圈

| 关键词 | 说明 |
|--------|------|
| `有机棉织带 宠物项圈` | 通搜 |
| `宠物项圈定制 小批量` | 成品项圈厂 |
| `织带厂 有机棉 定做` | 如果找不到成品厂，分开找织带+五金 |
| `宠物项圈 安全扣 breakaway` | 安全脱离扣 |

**重点地区**：温州（宠物配件）、广东东莞/深圳。

### 5. 三角巾（亚麻围巾/手帕）

| 关键词 | 说明 |
|--------|------|
| `亚麻三角巾 定制` | 通搜 |
| `亚麻围巾 小批量 定做` | 同类产品 |
| `服装配件定制 亚麻` | 如果找不到三角巾专门的厂 |

### 6. 肩带（有机棉宽织带）

| 关键词 | 说明 |
|--------|------|
| `有机棉织带 加宽 3.5cm` | 宽织带 |
| `帆布肩带 箱包配件` | 接近品类 |
| `棉质背包带 定制` | 成品肩带 |

## 四、供应商评估清单

联系后评估每一家，打 ✓ 或 ✗：

| 评估项 | 及格线 |
|--------|--------|
| **接受打样（2-3 件）** | ✅ 必须。拒绝的跳过 |
| **面料可自选或来料加工** | ✅ 必须。只用固定面料不选款的不行 |
| **法式缝（French seam）能做** | ✅ 必须。如果对方不知道什么是法式缝，换一家 |
| **样费** | ≤ $30/只 可接受 |
| **打样周期** | ≤ 2 周可接受 |
| **MOQ（批量起订）** | ≤ 100 件可接受。越低越好 |
| **沟通方式** | 微信 / WhatsApp 必须有一 |

## 五、排摸流程

```
第 1 步：1688 搜每个品类 → 标记 5-8 家看起来靠谱的
第 2 步：发同样一段询价话术给每家（模板见下）
第 3 步：收到回复后，筛掉不接打样的、沟通态度差的
第 4 步：对剩下的 2-3 家下单打样
第 5 步：收到样品后评估品质 → 选定供应商
```

### 询价话术模板

```
你好，我们是 Purrl Studio，一个宠物配饰品牌。
需要定制以下产品，请报价：

1. 亚麻梯形手提袋（猫用）
   - 尺寸：袋口 40cm，底部 36cm，高 40cm
   - 面料：100% 亚麻 220-260 gsm
   - 接缝：法式缝（内侧无毛边）
   - 配件：竹制提手 + 黄铜铆钉（我们提供或代采）
   - 数量：先打样 2 只，确认后首批 50-100 只

2. 如有现货亚麻面料可选，请发色卡。
   如无法式缝经验请提前告知，我们可换简化工序。

请回复：打样费、打样周期、MOQ、批量单价。谢谢。
```
