# Purrl — 产品深化设计：猫的舒适 × 人的舒适

> 2026-07-05 | 基于猫行为学与人体工学的产品设计

---

## 前置原则

Purrl 的产品不是"一个袋子装一只猫"。它是一个**人和猫共享的移动空间**。

设计的两条腿：
1. **猫的舒适** — 猫愿意待在里面，不抗拒，不焦虑
2. **人的舒适** — 人愿意提着出门，不累，不狼狈

两者冲突时，猫的需求优先。因为猫不会说"不舒服"，它只会挣扎或拒绝进入。

---

## 一、猫的舒适：科学视角

### 1.1 猫在运输中的核心需求

猫在户外运输时的应激（stress）来自于四个因素：

| 应激源 | 表现 | 设计应对 |
|--------|------|---------|
| **暴露感** | 猫感到无处隐藏，心率上升、瞳孔放大 | 袋体包裹身体，头可探出但身体受保护。亚麻不透明，给猫视觉屏障 |
| **不稳定感** | 脚底晃动，无法判断身体位置 | 软底贴合猫体曲线，猫自身的重量自然稳定袋子。避免硬板底部产生"漂浮"感 |
| **温度过热** | 封闭空间 + 猫体温 = 快速过热 | 亚麻是所有天然纤维中透气性最高的。敞开式设计让热量不积聚 |
| **气味陌生** | 新袋子有"非我"的气味，猫抗拒 | 亚麻本身无味（vs 涤纶有化学生产残留气味）。建议随袋附一张"猫信息素擦拭布"，让主人先在家让猫接触袋子 |

### 1.2 猫的体温管理

猫的舒适温度带是 **30-36°C**（人类是 20-25°C）。这意味着：

> 人觉得"刚好"的温度，猫已经觉得凉了。人觉得"有点热"的温度，猫才刚舒适。

但在封闭袋子里，猫自身的热量 + 外部温度叠加，很容易过热。

| 面料 | 透气性 | 对猫的影响 |
|------|:----:|------|
| **亚麻** | 最高 | 空气自由流通，猫不容易闷 |
| 有机棉厚织 | 中 | 可接受，但夏季需要选浅色袋身反射热量 |
| 涤纶/尼龙 | 极低 | 猫在袋内 15-20 分钟就会过热 |
| 羊毛毡 | 中低 | 冬季可用，夏季危险 |

> **设计决策**：亚麻猫袋不需要"透气网窗"——整只袋子本身就是透气的。这是亚麻作为猫袋材质的最大的功能优势，不是审美选择。

### 1.3 猫的安全感：半封闭 vs 全封闭 vs 全开放

| 类型 | 猫的感受 | Purrl 的对应 |
|------|---------|-------------|
| 全封闭猫包（飞机笼式） | 压抑、无法观察环境、加剧恐惧 | ❌ 不是 Purrl 的场景 |
| 全开放（抱在怀里） | 暴露、无处可逃 | ❌ |
| **半开放**（身体在袋内，头在外） | ✅ 最佳：身体受保护，视觉可观察，可随时缩回袋内 | ✅ Purrl 的袋型天然提供这个状态 |

猫在 Purrl 袋子里的行为模式：

```
猫探头 → 观察环境 → 觉得安全 → 继续探头
       → 觉得不安 → 缩回袋内 → 身体仍然被亚麻包裹 → 恢复安全 → 再次探头
```

这个循环是 Purrl 猫袋的核心体验——**猫自主控制暴露程度**。没有被封死、没有被强行抱出。

### 1.4 猫爪与面料

猫在袋内会踩奶（kneading），会用爪子抓挠袋身来标记领地。亚麻的应对：

| 面料 | 猫爪后果 |
|------|---------|
| 亚麻 | 会抽丝，但抽丝后的纹理反而有"生活感"。不会像涤纶一样勾出成片的拉丝 |
| 有机棉 | 会起毛球，但棉织物起球是自然的，不丑陋 |
| 涤纶 | 一勾就是一道永久性的亮丝线 |
| 尼龙 | 猫爪直接穿透，越抓越破 |

> **设计决策**：亚麻抽丝不是产品缺陷，是使用痕迹。可以做成品牌叙事——"Your cat's signature on the bag."

## 二、人的舒适：人体工学

### 2.1 携带方式对比

| 方式 | 受力部位 | 疲劳度 | 对猫的影响 |
|------|---------|:-----:|----------|
| **双提手手提** | 手掌 + 前臂 | 中 | 袋子相对稳定，猫不晃 |
| 单肩斜挎 | 单侧肩膀 + 脊椎 | 低（短途）/ 高（长途） | 袋子随走路晃动，猫不适 |
| 双肩包 | 双肩对称 | 最低 | 猫被完全封闭，无法观察 |

**Purrl 的选择**：双提手手提。不是最省力的，但是猫最稳定的。Purrl 的场景是"逛街"不是"徒步"——15-30 分钟的精致出门，不需要极致人体工学。

### 2.2 提手设计

| 参数 | 推荐值 | 原因 |
|------|:----:|------|
| 提手材质 | 竹制（原色/哑光） | 天然材质，手掌接触温润（vs 金属的冰冷/塑料的廉价） |
| 提手内径 | 12-14 cm | 女性手掌自然握持的舒适直径 |
| 提手截面 | 圆形，直径 18-22mm | 太细勒手，太粗捏不住 |
| 提手数量 | 2 个（对称） | 与参考照片一致，双手提持袋子平衡 |

### 2.3 肩带（预留第二代）

| 方式 | 优劣 |
|------|------|
| 可拆卸细肩带 | 解放双手，但袋子晃动，猫不舒服。可做选配但不做标配 |
| 不做肩带 | 第一代保持双提手纯粹。猫的稳定优先于人的省力 |

## 三、竞品拆解：当前猫出行产品的设计问题

### 3.1 SisiDogCouture 亚麻猫吊袋（$152）

| 做得好 | 做得差 |
|--------|--------|
| 100% 亚麻材质 | 单肩斜挎，猫随人走晃动 |
| 透气性好 | 袋型是"吊床"不是"袋子"，猫没有底部支撑 |
| 手工质感 | 设计感弱，像医疗辅助器材而非时尚配饰 |
| | 没有配套项圈/三角巾，穿搭不完整 |

### 3.2 亚马逊上的主流猫包（$20-60）

| 共同问题 |
|---------|
| 材质全是涤纶/尼龙，不透气，有化学味 |
| 都是"宠物运输工具"的视觉语言——网格窗、塑料卡扣、拉链 |
| 全封闭设计，猫看不到外面会加倍焦虑 |
| 人提着这些包出门——没有人会觉得它是好看的 |

### 3.3 设计师猫包（Pikore 皮革猫袋 ~$300）

| 问题 |
|------|
| 真皮材质，违背 Purrl 的品牌红线 |
| 硬底硬结构，猫在里面没有柔软包裹感 |
| 看起来好看但功能上跟 $30 的涤纶猫包没区别——全封闭、无透气 |

### 3.4 Purrl 的差异化

```
SisiDogCouture:     亚麻 ✅  审美 ❌  猫舒服 ⚠️
Amazon猫包:         便宜 ✅  其他全 ❌
Pikore皮包:         好看 ✅  材质 ❌  猫舒服 ❌

Purrl:              亚麻 ✅  审美 ✅  猫舒服 ✅  人好看 ✅
```

**Purrl 是目前全球唯一在"亚麻材质 + 审美在线 + 猫行为学考虑"三个维度上同时成立的产品。**

## 四、三角巾的功能设计

三角巾不是单纯的配饰——它在猫出行中有一个功能角色：

| 功能 | 说明 |
|------|------|
| 色彩锚点 | 室外环境中，一抹跟猫袋同色的三角巾让人一眼识别"这是一套 Purrl" |
| 安全感辅助 | 熟悉的织物接触猫的脖子，提供轻微压力（类似 ThunderShirt 原理），有镇定效果 |
| 身份标识 | 如果猫意外跑出袋子，三角巾上的 Purrl 徽标是辨识标记 |

---

## 五、项圈的安全设计

| 需求 | 解决方案 |
|------|---------|
| 卡住时自动脱离 | 黄铜安全脱离扣（breakaway buckle），受力 2-3kg 自动弹开 |
| 不会勒喉 | 织带宽度 1.2cm，分散压力 |
| 不刺激皮肤 | 有机棉亲肤，不含化学染料残留 |
| 跟袋子同色系 | 完整视觉搭配 |

## 六、产品细节清单（供打样参考）

| 细节 | 规格 | 为什么重要 |
|------|------|-----------|
| 袋身接缝 | 法式缝（French seam），内侧无毛边 | 猫不会勾到线头。亚麻散边严重，必须用封闭式缝法 |
| 底部加固 | 双层亚麻 + 十字交叉缝线 | 承重 5-6kg 不下坠 |
| 提手固定 | 黄铜铆钉 + 皮垫圈（有机棉织带替代皮） | 承重点需要分散压力，避免亚麻从铆钉处撕裂 |
| 内袋 | 一个小口袋缝在袋内侧 | 放猫零食、主人手机 |
| 边缘处理 | 亚麻卷边 + 双线缝 | 防止抽丝蔓延 |

```
正面：
"Linen you can feel. Brass that patinas. Nothing a cat wouldn't find in nature."

背面：
"Your cat will knead the fabric. You'll see little pulls in the linen.
That's not damage. That's your cat's signature on the bag.
Every Purrl Tote becomes one-of-a-kind."
```

把"亚麻抽丝"从产品缺陷重新定义为**猫和使用者之间的共同印记**。
