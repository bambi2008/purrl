## 同类品牌参考数据

- 2013 年预售：**$40,000**（纯靠朋友圈推荐）

- 波兰 Etsy 店，运营 8 年，812 条 reviews

- **年均 ~700-1,000 单** — 手工店天花板

## 三号客户画像

### 🥇 1号：城市审美派 Sophie

**她是谁**
> 32 岁，伦敦/纽约/柏林。建筑师、设计师、创意总监、时尚买手。
> 年收入 $95k-$150k。无孩，有一只猫。租房但装修极讲究。

**消费行为**
- 买 Cuyana 帆布包 $198 不眨眼，拒绝带 Logo 的衣服
- 用 Aesop、Our Place 的锅、Kaia 的瑜伽垫
- Instagram 审美克制，发猫像发建筑照片一样
- 每年给猫花 $1,200-2,500（远超 $653 平均线）

**为什么买 Purrl**
> "我给猫买过三条项圈都是凑合的。Purrl 终于让我觉得——我和我的猫是搭配过的。"

- 她买的是 "我和猫一起出门的完整造型"，不是猫包
- 材质故事说服她（亚麻、有机棉、可降解）
- 建筑师的克制美学跟她是同频的

**不买的原因**
- 产品图不好看 → 第一眼就划走
- 材质不明确 → 她会读面料成分表
- Logo 太大或文案太腻 → 拒绝任何喧哗的品牌

**触达渠道**
- Instagram（精选、黑白建筑感构图）
- Pinterest（跟 Cuyana / 家居 board 在一起）
- 设计类 newsletters（Dezeen、ArchDaily）
- 朋友推荐（她的社交圈审美相近）

### 🥈 2号：环保信仰派 Emma

**她是谁**
> 28 岁，旧金山/波特兰/柏林。产品经理、NGO 从业者、可持续发展顾问。
> 年收入 $80k-$120k。素食主义者或弹性素食。有一只领养猫。

**消费行为**
- 主动查 GOTS 认证、OEKO-TEX、B Corp
- 买 Girlfriend Collective、Patagonia、Allbirds
- 拒绝一切塑料包装
- 每年给猫花 $1,000-2,000

**为什么买 Purrl**
> "终于有一个猫品牌不用塑料、不用皮革，材料表我读得懂。"

- 品牌红线（纯环保材料、永不出现真皮）跟她完全一致
- "100% natural. 100% biodegradable" 就是转化按钮
- 愿意为环保溢价多付 20-30%

**不买的原因**
- 没有认证（GOTS、OEKO-TEX）→ 她觉得是漂绿
- 价格不透明 → 她想知道为什么 $128
- 包装用塑料 → 直接退货

**触达渠道**
- Google 搜索 "sustainable cat carrier"
- 环保博主推荐
- Patagonia / Allbirds 用户社区

### 🥉 3号：宠溺家长派 Linda

**她是谁**
> 41 岁，东京/香港/新加坡。投行 VP、律所合伙人、科技公司高管。
> 年收入 $200k+。无孩或孩子已上大学。养了两只纯种猫。

- 需要猫袋看起来像手袋而不是宠物运输工具

**不买的原因**
- 没有品牌信任背书（没在 Vogue / 设计媒体上见过）

**触达渠道**
- 设计杂志（Vogue Living、Architectural Digest）
- 免税店 / 高端买手店
- 朋友社交圈

## 购买路径（Sophie 视角）

```
发现（IG/朋友推荐）→ 感兴趣（看官网 3 秒定生死）
→ 验证（读材质说明、看真实买家图、搜 IG 口碑）
→ 购买（Stripe/Apple Pay、$128 在她冲动区间内）
→ 传播（发 IG Story、2 个朋友 DM 问链接）
```

## 关键内容策略

| 指标 | 目标 |
|------|:----:|
| IG 互动率 | >3%（时尚平均 1-1.5%） |
| 官网访问→购买转化率 | >2.5%（DTC 平均 1-2%） |
| 首月订单 | 30-50 单 |
| UGC 自发率 | >15% 买家发帖 |
| 客单价 | $128+（单品）/$156（套装） |

### 品牌红线
1. **纯环保材料** — 全线天然可降解，永不出现真皮/素皮
2. **美** — 建筑师的克制美学，材质/比例/留白
3. **不教条** — 不讲环保说教，用产品和设计本身说话
