# Purrl — 用户流程框架（A）

> 2026-06-28 | 核心智慧：先用 → 看到自己猫变美 → 才注册

---

## 第一次访问（核心转化路径）

```
① 落地页
   惊艳呈现(写真/短视频轮播) + "Every cat is a Purrl" + 免费试一次
        ▼
② 上传猫照片(零门槛，不强制注册)
        ▼
③ 即时反馈(秒级，制造惊喜)
   "你的猫：暹罗 · 奶油白+巧克力色 · 优雅型"
   ← CatProfile色板+气质先出(便宜快速)
        ▼
④ 免费产出"哇"时刻
   一张AI写真(带水印) 或 一段5秒短视频(带水印)
        ▼
⑤ 三条变现分叉
   分享(带水印拉新) / 解锁高清(3.9元首单) / 看更多(引导注册)
        ▼
⑥ 注册(此时才要求——已经有了"我的猫")
        ▼
⑦ 深度变现：写真集订阅 / 配饰下单 / 3D雕塑
```

## 关键设计决策

| 决策 | 选择 | 理由 |
|------|------|------|
| 注册时机 | 第④步之后 | 先给价值再要注册 |
| 第一个免费产出 | 写真 or 短视频 | 制造"哇"时刻=留存起点 |
| 首单价格 | 3.9元解锁高清 | 跨过"从没付过费"心理坎 |
| 水印策略 | 免费产出带Purrl水印 | 每次分享=免费广告=获客闭环 |

## 回访用户（已有CatProfile）

```
打开 → 直接进"我的猫"主页
   ├─ 新写真  ├─ 短视频  ├─ 逛配饰  ├─ 3D雕塑  └─ 和猫聊天
每次回访加深与"我的猫档案"的绑定
```

## 完整漏斗（对齐v3协同路径，转化率均为假设待验证）

```
落地页访客
   │ 上传率 ~30-50%
   ▼
免费产出→"哇"时刻
   │ 分享率 ~10-20% ──→ 带新访客(病毒循环)
   │ 首单转化 ~5-10%
   ▼
3.9元首单用户
   │ 配饰转化 ~15-20%
   ▼
配饰复购用户(利润核心)
   │ 雕塑转化 ~5%
   ▼
3D雕塑高客单用户
```

## 核心智慧

```
传统：先注册 → 再用 → 流失90%
Purrl：先用 → 看到猫变美 → 才注册 → 已被钩住

洞察：用户不为"注册"而来，为"看我的猫变美"而来。
把注册推到价值兑现之后。
```
