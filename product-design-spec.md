## 一、The Purrl Tote（猫袋）

### 设计概述

参考原型照片：软结构敞口猫袋，上窄下宽梯形，猫坐入后头部和前爪伸出袋口。竹制圆形提手。第一代不做结构改动，仅将织物从白色针织替换为 100% 亚麻。

### 尺寸

| 部位 | 尺寸 | 说明 |
|------|:----:|------|
| 袋口宽度 | 28-30 cm | 猫头+双前爪可伸出，身体不会滑出 |
| 底部宽度 | 35-38 cm | 猫有足够空间转身和坐稳 |
| 深度（袋高） | 38-42 cm | 覆盖猫的体长+颈部露出 |
| 提手直径 | 12-14 cm | 竹制圆环，手握尺寸 |
| 提手内距 | 开口自然垂坠时约 20-22 cm |

适配猫体重：**≤ 5.5 kg（12 lbs）**

### 版型

```
        ┌── 28-30cm ──┐
        │  猫头+前爪伸出  │
        │               │
       ╱                 ╲       ← 两侧自然斜裁
      ╱                   ╲
     │                     │
     │    猫身体在袋内    │       ← 软结构，猫体撑开袋子
     │                     │
     └───── 35-38cm ──────┘
         上窄下宽梯形
```

- 版型直接沿用参考照片包的剪裁（梯形软袋）
- 不做内衬骨架，靠亚麻织物自身的垂坠感保持形状
- 底部无硬板，软底加厚双层亚麻

### 材质

| 部位 | 材料 | 说明 |
|------|------|------|
| 袋身 | 100% 欧洲亚麻，220-260 gsm | 中等厚织，透气不透明 |
| 提手 | 天然竹制圆环，直径 13cm | 哑光处理，原色（浅棕） |
| 五金 | 黄铜（brass）扣环 ×2 | 连接提手与袋身，哑光 |
| 内衬 | 100% 有机棉薄织，180 gsm | 亚麻 + 棉内层，双层加固 |
| 缝线 | 亚麻线（同色系） | 天然材质一致 |
| Logo | 纯棉标签，烫印 Purrl | 缝在袋内侧 |

### 配色

| 配色名 | 袋身 | 提手 | 五金 |
|--------|------|------|:---:|
| **Natural Flax** | 亚麻本色（暖灰米） | 原色竹 | 哑光黄铜 |
| **Warm Linen** | 亚麻焦糖色（天然染） | 原色竹 | 哑光黄铜 |
| **Stone White** | 亚麻本白（未漂白） | 原色竹 | 哑光黄铜 |

> 第一代先做 Natural Flax 单色。后续两个配色根据市场反馈迭代。

## 二、Linen Bandana（三角巾）

### 设计概述

纯亚麻三角巾，套入式设计（滑入项圈上方，不用卡扣）。跟猫袋同色系，视觉上统一。三款自然色可选，匹配不同毛色的猫。

### 尺寸

| 部位 | 尺寸 |
|------|:----:|
| 三角形底边 | 28 cm |
| 三角形腰边 | 22 cm |
| 套口宽度 | 3.5 cm（可过标准项圈） |

### 材质

| 部位 | 材料 |
|------|------|
| 面料 | 100% 亚麻，180-200 gsm |
| 套口 | 同面料双层折叠，无需弹力带 |

### 配色

| 配色名 | 色值 |
|--------|------|
| **Fog Grey** | 暖灰（跟 Natural Flax 猫袋匹配） |
| **Golden Sand** | 浅赭石（亚麻天然染） |
| **Cloud White** | 本白（未漂白亚麻本色） |

## 三、Cotton Collar（项圈）

### 设计概述

有机棉织带项圈，黄铜扣件，安全卡扣设计。无塑料部件。跟猫袋+三角巾形成完整的搭配系统。

### 尺寸

| 部位 | 尺寸 |
|------|:----:|
| 调节范围 | 20-30 cm（适合大多数中大型猫） |
| 织带宽度 | 1.2 cm |
| 厚度 | 2 mm |

### 材质

| 部位 | 材料 |
|------|------|
| 织带 | 100% GOTS 有机棉，斜纹编织 |
| 主扣 | 黄铜 D 环 + 安全脱离扣（breakaway buckle） |
| 调节扣 | 黄铜日字扣（tri-glide） |
| 品牌环 | 黄铜小圆环，激光刻 Purrl |

### 配色

| 配色名 | 织带色 | 五金 |
|--------|------|:---:|
| **Flax** | 亚麻本色（暖灰米） | 哑光黄铜 |
| **Bark** | 树皮棕（天然染） | 哑光黄铜 |
| **Bone** | 骨白 | 哑光黄铜 |
