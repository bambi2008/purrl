# Purrl — 商业测算 v3（加速起势版）

> 2026-06-28 | 修正：前3个月付费广告测LTV:CAC，测通即放量，跳过"慢慢爬"阶段

---

## v2为什么慢

```
v2的错误假设：
  第一年：自然增长 2,000 用户
  广告费 $6,000 → 获客 2,000 人 → CAC $3
  但这些人消费浅(无配饰复购积累)，第一年净利 $472

  问题是：既然CAC只要$3，那为什么不多投？
  LTV能到多少？

v2没有回答的问题：
  LTV:CAC 是多少？如果它是>3，应该第一年就烧$50k。
  如果它<1，那广告根本不该投。
```

---

## v3 关键修正：按LTV:CAC决策，不按"自然增长"规划

### 先测LTV:CAC（前3个月）

```
目标：花$3,000 买 1,000 个付费用户（CAC $3）
跟踪12周：
  看他们是否订阅($24/年) → 看是否买配饰($29) → 看是否复购
  算出平均 LTV at Month 6
```

三种可能的结果：

| 场景 | LTV:CAC | 决策 |
|------|---------|------|
| 🟢 健康 | >3 | 第一年就放量，$50k广告砸下去 |
| 🟡 可接受 | 1.5-3 | 谨慎投放，优化价格/漏斗 |
| 🔴 不健康 | <1 | 调定价或放弃广告，转自然流量 |

**v3假设LTV:CAC = 4（健康）**，基于：

- 付费用户订阅$24 + 67%续约 = 首年LTV $59
- 其中30%买配饰$29 = +$8.7
- 其中8%买人猫共生$189 = +$15
- 总LTV ≈ $83
- CAC $3 → LTV:CAC = 28x（夸张不）

合理假设LTV:$40-60区间，CAC$3-5，LTV:CAC ≈ 8-15x。v3取保守值4x。

### 第一年——投入期（广告放量，净利为负）

```
用户获取：
  Q1: 花$3,000测LTV:CAC——确认健康
  Q2-Q4: 每月花$10k-$15k广告买量
  
用户基数：
  全年新增注册用户：10,000人
  总广告费：$40,000
  CAC：$4/注册
  
  付费用户(10%转化)：1,000人 × $24 = $24,000
  续费：0(第一年)

配饰收入(首单)：
  1,000×20%=200件 × $29 = $5,800

配饰复购(老用户—年底才开始有)：
  假设Q4有200老用户：200×15%×1次=30件×$29=$870

单次收入：
  约25%免费用户试单次：2,500×$1.99=$4,975

3D雕塑：
  1,000×5%=50件×$69=$3,450

人猫共生：
  1,000×8%=80件，均价$189=$15,120

--- 年度总计 ---
总收入：$54,215

成本：
  AI API：~$500
  广告费：$40,000
  POD履约(200+30+50+80=360件)：≈$5,500
  支付手续费(3%)：~$1,600
总成本：~$47,600

净利润：$6,615
``` 

（比v2好一倍，但还没起飞——第一年主要是积累用户基数）
