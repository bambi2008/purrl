## 三、判断标准

| 信号 | 含义 |
|------|------|
| 单帖 < 20 赞，0 评论 | ❌ 图片不够好看或者触达不到 Sophie |
| 单帖 30-50 赞，有人问"where to buy" | ⚠️ 有信号，但不够强 |
| 单帖 > 50 赞 + 有人留言问购买 + 有 3+ 人点 bio 链接 | ✅ 强烈信号，Sophie 存在 |
| 有人 DM 问价格 / 材质 / 什么时候上线 | ✅ 最强信号，用户信任程度高 |

## 补充：养号日程（自然增长）

> 核心原理：IG 新号自然触达率极低。不是靠发帖等人来，而是主动出现在 Sophie 面前。

### 每日操作（15-20 分钟）

| 动作 | 数量 | 怎么做 |
|------|:----:|------|
| 主动关注 | 15-20 人 | 搜 Cuyana / not PERFECT LINEN / Mansur Gavriel → 看评论区 → 关注评论者（不是关注品牌） |
| 评论互动 | 3-5 条 | 在目标品牌的帖子里留自然评论。不提 Purrl。头像和用户名就是广告 |
| 点赞 | 20-30 个 | 在目标用户自己的帖子里点赞，让头像出现在他们的通知栏 |
| 发帖 | 每周 2-3 条 | 审美内容，不是营销帖 |

### 2 周增长预期

```
第 1 周：0 → 100-150 关注者（回关率 30-40%）
第 2 周：150 → 300-400 关注者（自然增长开始启动）
第 3-4 周：第一条帖 > 50 赞，Bio 链接开始有人点 → 等待列表页收邮箱
```

### 关键规则

| ✅ 做 | ❌ 不做 |
|------|--------|
| 每天只关注 15-20 人（IG 限制大量关注） | 一天关注 100+（被标记为机器人） |
| 评论像真人——具体、有趣 | "Great post!" "Love your page!" |
| 头像放 Purrl 产品图 | 不要在评论里推销 Purrl |
| 每天固定时间操作（培养算法习惯） | 三天打鱼两天晒网 |
| 关注 Sophie 是哪些人：关注了 Cuyana / NPL / Mansur Gavriel / Dagne Dover 且头像里有猫的用户 | 关注宠物大号、搞笑号、跟 Purrl 审美无关的人 |
