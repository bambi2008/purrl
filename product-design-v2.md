# Purrl — 产品设计 V2（基于深化研究）

> 2026-07-05 | V1 的 7 个猫行为学问题 → 7 个设计修改

---

## 一、逐个修改

### 1. 侧开口 ✅

**问题**：猫被从上方放入时，模拟了鸟类捕食攻击。AAFP/ISFM 指南建议猫偏好侧入口。

**方案**：袋身侧边加隐藏式侧开口。

- 位置：袋子右侧（面向猫主人的右侧），从袋口往下约 10cm 处开始
- 开口方式：亚麻重叠片 + 隐藏磁吸扣（双层亚麻包裹磁性薄片，外面看不到任何塑料/金属）
- 宽度：约 20cm，足够一只猫走进去
- 日常使用：侧开口默认闭合，外观上看不出。需要装猫时，打开侧开口，猫自己走入。然后扣上磁吸，猫从上方探头

**对人来说的意义**：一个人可以把袋子放在地上，打开侧开口，猫走进去，扣上磁吸，提起来——全程一个人完成。

---

### 2. 围栏翻边 ✅

**问题**：猫在户外需要"可看可藏"的切换。敞口让猫完全暴露，缩回袋内又完全看不见。

**方案**：袋口内侧加 10cm 高的亚麻翻边。类似 cowl-neck top 的领子。

- 翻下来：袋口跟现在一样高，猫正常探头
- 翻上去：猫脖子周围多了一圈立起来的亚麻。猫的身体被围住，但头仍然在外面，眼睛能看到周围

**不是"盖住"**，是给猫脖子一圈。猫生理学上，猫的脖子和下巴是气味腺密集区——围住这一圈，猫会本能地觉得安全，同时又没有封闭的压抑感。

---

### 3. 薄亚麻底板垫 ✅

**问题**：软底对曲线贴合好但对平衡感不好。猫在晃动的底盘上需要持续微调姿势，腹部疲劳。

**方案**：底部放一片可移除的薄亚麻垫。

- 两层亚麻夹一层薄有机棉絮（3mm 厚，类似帆布袋底板但更软）
- 垫子可取出来让猫在家先闻、先睡——建立气味关联
- 垫子放入袋子后，猫的重力压出形状，走路时不会晃动
- 可机洗

---

### 4. 秋冬方案 ✅

第一代只做春夏秋。秋冬款以后推出毛毡版本。

---

### 5. 可拆卸内胆 ✅（调整方案）

**问题**：猫可能排泄，内衬需要清洗。但不解决水渗透的问题。

**方案调整**：不做可拆卸内胆。做**全袋可机洗**。

- 亚麻面料在生产时预先水洗 2 次（pre-shrink），去除缩水率
- 袋身整体可以放入洗衣袋，冷水轻柔机洗
- 竹提手和黄铜扣件可拆卸（用隐藏式套扣固定，不是铆钉）

做不到 100% 防水（天然材料做不到），但能 100% 清洁。猫尿发生后，整个袋子进洗衣机。

---

### 6. 安全绳 ✅

**问题**：敞口猫袋没有束缚，猫受惊可能跳出。

**方案**：项圈 D 环 + 袋内安全绳锚点。

- 项圈上已有 D 环（产品设计规格已有）
- 袋子内侧底部缝一个微型黄铜环做锚点
- 附带一根 35cm 长的亚麻编织安全绳，两端是小号黄铜扣钩
- 平常收纳在袋内小口袋中。需要时挂上：一端扣项圈 D 环，一端扣袋子底部锚点
- 绳子长度刚好够猫探头和缩回，不够猫跳出

---

### 7. 斜挎肩带 ✅

**问题**：双提手长时间单手会累。

**方案**：可拆卸斜挎肩带作为标准配件。

- 材料：有机棉编织宽带（宽 3.5cm，分散压力）
- 两端黄铜扣钩，连接在竹提手内侧预留的隐藏黄铜挂环上
- 收入袋内小口袋。不挂时看不见，挂了是可调节长度的斜挎

## 三、V2 版型示意

```
         ┌──── 28-30cm ────┐  ← 袋口
         │  翻边（可翻上翻下）│
  ┌──────┤                  ├──────┐
  │ 侧开口│  猫头+前爪伸出     │      │  ← 右侧：磁吸重叠片
  │(磁吸) │                  │      │     一个人可以从这里装猫
  └──────┤                  ├──────┘
         ╲                  ╱
          ╲                ╱
           │              │
           │  猫身体      │
           │  + 底板垫    │
           │  + 安全绳锚点 │
           └──────────────┘
           ┌── 35-38cm ──┐
               竹提手 + 隐藏挂环
               （可挂斜挎肩带）
```

- 提手竹环内侧各挂一个黄铜迷你环——平时看不见，需要时挂斜挎带
- 侧开口在袋子右侧（人用右手打开），猫从右侧走入
