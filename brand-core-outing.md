# Purrl — 品牌内核：出门美学

> 2026-06-28 | 一个场景：精心搭配好的猫，和主人一起出门

---

## 品牌场景

```
主场景只有这一个：

  你背上 Purrl 猫袋托特
  猫戴着同色系项圈/领结
  猫窝在包里探出头
  你们一起出门了

  你穿什么不重要。
  重要的是你的猫和它的包、它的项圈是搭配过的。
  别人一看就知道：这只猫是一整套 Purrl。

不是"猫包"。不是"猫项圈"。
是"出门的猫和它这一身的搭配"。
```

---

## 产品线（围绕场景倒推）

出门需要三件东西：

| 出门需要 | Purrl 产品 | 售价 | 角色 |
|---------|-----------|------|------|
| 猫坐在里面 | **猫袋托特** | $129 | 旗舰：猫出门的姿态 |
| 猫戴着装饰 | **项圈/领结/三角巾** | $39-49 | 搭配：和包同色系 |
| 猫挂着保险 | **牵引背带** | $59 | 配件：防跑+和安全带扣 |

三个单品，**一套配色**，一次出门。

---

## 场景化定价

```
场景定价：
  包+项圈 = $129+49 = $178
  包+项圈+背带 = $129+49+59 = $237

不推单品，推"出门套装"：
  "The Purrl Outing Set — bag + collar + harness, matched to your cat."
  套装价 $179（省$58）
```

---

## 识美驱动这个场景

```
用户第一次来：
  ① 上传猫照片
  ② 识美输出色板："你的猫：暖杏色+象牙白，琥珀眼"
  ③ 推荐出门套装："根据你的猫的毛色 → 推荐 焦糖棕套装"
  ④ 用户看到预览：猫照片上叠了包+项圈的效果图
  ⑤ 一键下单
```

**用户不是买包。是买"我的猫搭配好的出门方案"。**
