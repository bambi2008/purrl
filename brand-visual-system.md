# Purrl — 品牌视觉系统（D）

> 2026-06-28 | 建筑师的眼光：比例 · 材质 · 光影

---

## 品牌基因

Purrl 不是宠物用品品牌。是**用建筑师的眼光看猫**的品牌。

```
❌ 可爱、圆润、粉色、花朵 → 淘宝风
✅ 干净、克制、留白、材质 → 建筑感
```

---

## Logo

**方向 B：符号 + 文字**

一个极简的圆 + 弧线 = 猫脸 + 一颗珍珠，暗示 Purrl = Purr + Pearl。

```
┌────────────────────────────┐
│  ◕ )  Purrl                │
│  (几何圆=珍珠/猫脸)         │
│  Every cat is a Purrl.      │
└────────────────────────────┘
```

*纯文字版（MVP先用）：Inter Tight 字体，字母 r 可有耳部变形*

## 视觉调性三条红线

| 红线 | 解释 |
|------|------|
| 不卖萌 | 不用圆角气泡、花瓣、猫爪印 |
| 留白 | 每页留白 > 30%，信息密度低 |
| 无滤镜 | 产品图只用自然光，不用粉红/暖黄滤镜 |
