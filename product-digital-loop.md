# Purrl — 出门场景 × 数字产品的闭环

> 2026-06-28 | 相册/视频/3D模型不是独立产品，是出门体验的延伸

---

## 核心逻辑

用户不是来买相册的。用户是来买"我的猫出门了"这件事的。

数字产品的角色：

```
出门前：识美分析毛色 → 推荐配色方案 → 选购包+配饰
出门中：拍照片/视频 → 猫戴着Purrl出门的实拍
出门后：AI写真/AI视频/3D模型 → 把这次出门变成作品保存下来
```

**数字产品不是获客工具。是实物的增值服务。**

---

## 每件数字产品在场景中的位置

### ① AI写真（出门纪念照）

```
用户买了包+项圈 → 带猫出门 → 拍了几张照片
                                 │
                                 ▼
用户把照片上传到 Purrl
  → AI转成艺术风格（印象派/浮世绘/极简）
  → 带 Purrl 水印的免费版可以发社交
  → 高清无水印版需要是订阅会员

不是"AI写真工具"。
是"你带猫出门了，给你一张纪念照"。
```

### ② AI短视频（出门动态）

```
同样的照片/视频
  → AI生成 5 秒猫眨眼/摇尾巴
  → 免费版带水印，分享到 Ins/TikTok = 免费广告
  → 高清无水印版会员专属

不是"短视频工具"。
是"你带猫出门了，让它在社交媒体上动起来"。
```

### ③ 3D模型（出门留念）

```
用户在 Purrl 上已经有猫的识美数据
  → 一键生成 3D 旋转模型
  → 网页上可以旋转查看
  → 可以下单打印成实物雕塑（$69，放在家里）

不是"3D工具"。
是"你带猫出门的记忆，变成放在桌上的东西"。
```

## 用户旅程（完整版）

```
① 听说 → 朋友在Ins分享了一张猫的惊艳写真（带Purrl水印）
② 尝试 → 打开Purrl网站，上传自己猫的照片
③ 识美 → "你的猫是英短蓝猫，灰蓝色+琥珀眼"
④ 种草 → "推荐出门套装：雾灰蓝项圈+同色系猫袋"（预览图）
⑤ 下单 → $129包+$49项圈 = $178
⑥ 收货 → 给猫戴上，背上包，出门
⑦ 记录 → 拍照片/视频，上传到Purrl，生成艺术写真/短视频
⑧ 分享 → 发Ins/TikTok → 朋友看到又回到①

▸ 实物赚钱，数字传播。
▸ 没有数字层，实物没人知道。
▸ 没有实物，数字层没有根基。
```
