# Purrl — 人猫共生 · 移动共享空间

> 2026-06-28 | 变革式思考：不是"猫坐的车"，是"人和猫一起逛街的移动工具"

---

## 已有方案的问题

```
婴儿车改造的猫推车：
  ❌ 猫关在篷子里，看不见外面，和航空箱没区别
  ❌ 巨大笨重，像推着婴儿车，社交尴尬
  ❌ 和人的互动为零——你推你的，猫待它的

核心矛盾：
  你要带猫出门，不是为了"运输"猫。
  是为了和猫一起体验外面。
```

---

## 变革设计

### 不是猫推车，是移动共享座舱

```
结构分解：

  下层：置物篮（30L）——逛街买东西放进去
  上层：猫的半开放座舱——猫可以站起来探头
  中间：空的——你可以低头看到猫，猫可以搭爪子在你手上

  你推着走的时候：
    → 猫探出头看街景
    → 你随时摸它的头
    → 路人看到会笑
    → 这不是运输，是和猫一起逛街
```

### 建筑师的设计点

| 维度 | 市面产品 | Purrl 设计 |
|------|---------|-----------|
| 比例 | 像婴儿车，巨大笨重 | 窄体+低重心，像 Muji 购物车+猫座 |
| 材质 | 黑色尼龙+银色钢管 | 阳极氧化铝管(轻)+编织帆布座舱(透气可拆洗)+静音轮 |
| 猫和人的关系 | 猫被罩住，人看不见猫 | 半开放设计，视线互通 |
| 折叠 | 很少能折 | 一键折叠，放入轿车后备箱 |
| 收纳 | 无 | 下篮30L，周末买菜+遛猫一次搞定 |

### 使用场景

```
周末上午：
  你推着车去 farmers market
  上层：猫探出头看蔬菜摊
  下层：你买的西红柿和面包
  你一边走一边和猫说话

日常傍晚：
  不出门购物，只推着猫在小区散步
  它探头看树上的鸟，你慢慢走
  不用弯腰抱它，它在车上
```

- 普通猫推车 $80-120（丑，但便宜）

## 为什么这是建筑师该做的

```
普通产品设计师做猫推车：
  → 研究宠物用品市场
  → 参考猫推车竞品
  → 做得更好看一点

建筑师做猫推车：
  → 研究人和猫在公共空间的关系
  → 研究一辆车在城市中移动的姿态
  → 研究铝管的弯曲角度和织物的垂坠感
  → 做出来的东西是"城市家具"，不是"宠物用品"
```
